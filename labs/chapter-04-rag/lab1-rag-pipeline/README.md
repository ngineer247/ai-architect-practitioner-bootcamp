# RAG Pipeline

**Chapter 04 — The AI Architect & Practitioner Bootcamp**

## What You Build

Build an end-to-end RAG pipeline from query to grounded answer.

## Quick Start

```bash
pip install -r requirements.txt
python rag_pipeline.py
```

## Tasks

1. Run the quick start command and review the output
2. Modify one configuration parameter and observe the change
3. Write one additional test case in `tests/`
4. Connect to a real model endpoint and run against live output
