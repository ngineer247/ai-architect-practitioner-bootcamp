# Evaluation Harness

**Chapter 15 — The AI Architect & Practitioner Bootcamp**

## What You Build

Run LLM-as-judge evaluation with safety gate and CI/CD release decision.

## Quick Start

```bash
pip install -r requirements.txt
python evaluator.py
```

## Tasks

1. Run the quick start command and review the output
2. Modify one configuration parameter and observe the change
3. Write one additional test case in `tests/`
4. Connect to a real model endpoint and run against live output
