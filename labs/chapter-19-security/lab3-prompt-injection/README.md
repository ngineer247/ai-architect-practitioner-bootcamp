# Prompt Injection Test Suite

**Chapter 19 — The AI Architect & Practitioner Bootcamp**

## What You Build

Run 10 typed red-team injection cases against your AI system.

## Quick Start

```bash
pip install -r requirements.txt
python run_tests.py
```

## Tasks

1. Run the quick start command and review the output
2. Modify one configuration parameter and observe the change
3. Write one additional test case in `tests/`
4. Connect to a real model endpoint and run against live output
