# Basic Agent Loop

**Chapter 07 — The AI Architect & Practitioner Bootcamp**

## What You Build

Implement the fundamental decide→act→observe agent loop with tool authorization.

## Quick Start

```bash
pip install -r requirements.txt
pytest tests/ -v
```

## Tasks

1. Run the quick start command and review the output
2. Modify one configuration parameter and observe the change
3. Write one additional test case in `tests/`
4. Connect to a real model endpoint and run against live output
