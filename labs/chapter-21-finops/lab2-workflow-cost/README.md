# Workflow Cost Collector

**Chapter 21 — The AI Architect & Practitioner Bootcamp**

## What You Build

Aggregate cost events into per-workflow, per-tenant cost-per-task reports.

## Quick Start

```bash
pip install -r requirements.txt
python workflow_cost.py sample_events.jsonl
```

## Tasks

1. Run the quick start command and review the output
2. Modify one configuration parameter and observe the change
3. Write one additional test case in `tests/`
4. Connect to a real model endpoint and run against live output
