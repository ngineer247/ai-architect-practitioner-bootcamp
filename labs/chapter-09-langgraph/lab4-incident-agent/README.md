# LangGraph Incident Agent

**Chapter 09 — The AI Architect & Practitioner Bootcamp**

## What You Build

Build a stateful incident investigation agent with HITL using PostgresSaver.

## Quick Start

```bash
pip install -r requirements.txt
python incident_agent.py
```

## Tasks

1. Run the quick start command and review the output
2. Modify one configuration parameter and observe the change
3. Write one additional test case in `tests/`
4. Connect to a real model endpoint and run against live output
